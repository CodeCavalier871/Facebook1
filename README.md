# Facebook账单户功能详解与使用指南
## 一、基础定义
**账单户核心职能**  
作为Facebook官方广告资金结算单元，承担广告费用归集与支付清算职能。系统自动生成周期滚动账单（通常按自然月），触发结算条件包括：
- **阈值触发**：当月累计消耗达500美元自动扣款  
- **周期触发**：月底清零未达阈值的账户余额  
## 二、账户矩阵体系
### 层级权限架构
| 账户类型       | 创建权限      | 管理数量上限 | 支付方式               |
|----------------|---------------|--------------|------------------------|
| 企业主账户     | Facebook官方  | 不限         | 信用卡/代理商预付      |  
| 子运营账户     | 主账户授权    | 50个/BM      | 限额共享/独立绑定支付  |
### 账户限额类型
- **新号保护期**：初始50美元/日消耗限额
- **信用额度扩容**：连续21天稳定投放可升250美元/日
- **企业白名单**：提交公司资质后解除日限额
## 三、功能全景
### 数据中枢模块  
- **多维度消耗看板**  
  支持实时追踪分广告组/地域/时段消耗数据颗粒度达分钟级
- **跨境支付适配**  
  兼容42种货币结算，自动按当日汇率转换
- **账单审计系统**  
  可追溯180天内每笔扣费对应具体广告及用户互动行为
### 风险控制模块
- **三轮预警机制**  
  1. 预算消耗达80%弹窗提醒  
  2. 单日消耗超限自动暂停  
  3. 异常点击智能拦截（精确度98.7%）
